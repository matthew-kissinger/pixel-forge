Sprite atlas — 60-free-plants

60 plant sprites (each pre-downscaled to 512px max) packed into a single 4096² POT atlas via the kiln shelf-packer. Each gridline rectangle corresponds to one frame entry in the JSON sidecar.

Survival-kit FBX ingest

80 Kenney survival-kit FBX models converted to GLB via three.js FBXLoader + GLTFExporter, then tightened through @gltf-transform dedup + prune. Texture references were stripped because the FBXs point to external .png files we don't resolve — material colors are preserved, so a downstream retex pass can restore surfaces.
